# Telegram Bot for n8n Workflow Sales

This is a Telegram bot designed to automate the sale of n8n workflows (JSON configurations for server monitoring automation).

## Project Overview

The bot acts as a full-fledged online store within Telegram, offering various n8n workflows for server monitoring. Key features include:
- Watermarking of purchased workflows to identify buyers.
- Protection against leaks.
- Future updates for purchased workflows.

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Jade-Dragon88/workflow_shop_bot.git
cd workflow_shop_bot
```

### 2. Create and activate a virtual environment (recommended)
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure environment variables
Copy the example environment file and fill in your details:
```bash
cp .env.example .env
```
Edit the `.env` file with your `BOT_TOKEN`, `SUPABASE_URL`, `SUPABASE_KEY`, `YUKASSA_TOKEN`, `ADMIN_IDS`, and `PRIVATE_CHANNEL_ID`.
